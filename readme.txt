PostFlow AI is a powerful workflow-driven WordPress plugin developed by UfirstDEV Technologies to automate content creation at scale. It helps bloggers, agencies, and content-driven businesses generate high-quality posts with zero manual effort. Upload your topic list, set categories and posting schedule, and PostFlow AI will handle the rest—publishing fresh, realistic content in your brand tone automatically or on demand.

== Description ==

**PostFlow AI** enables WordPress creators to automate content creation effortlessly. With built-in license and subscription validation, usage tracking, flexible time-based triggers, and secure REST integration, PostFlow AI ensures every post is generated, published, and tracked with precision. 

**Key Features:**
* Upload and manage a list of topics and categories
* Automatic or on-demand content generation and publishing
* Realistic, brand-toned content
* License and subscription validation (Free, Pro, Scale)
* Usage tracking and post credits management
* Flexible time-based triggers: daily, hourly, minutely, or manual
* Dashboard with statistics for pending and published content
* Secure REST communication with your WordPress site
* Multi-user and multi-site compatible

== Installation ==

1. Upload the `postflow-ai` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the PostFlow AI Settings page.
4. Enter your license key to activate your subscription plan.
5. Upload your topic list (CSV/Excel) and configure categories and posting schedule.
6. Set the time-based trigger or manually trigger content generation.

== Frequently Asked Questions ==

= Can multiple users trigger content at the same time? =
Yes. PostFlow AI handles multiple simultaneous requests safely, tracking each execution and credit usage independently.

= Which subscription plans are supported? =
PostFlow AI supports Free, Pro, and Scale plans. Each plan has a monthly post limit verified via license key.

= Can I schedule posts automatically? =
Yes. You can choose daily, hourly, minute-based intervals, or trigger content manually.

= How is content quality ensured? =
Content is generated in your brand’s tone based on the topic list you provide. Each article is realistic and ready for publishing.

= Is my site secure while using PostFlow AI? =
Yes. All data communication uses WordPress REST API with secure credentials. License verification and usage tracking are also secure.
